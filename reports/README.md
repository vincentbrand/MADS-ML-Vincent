# Reports

One report per chapter. **All prose is written by the student** — these `.tex`
files are structural scaffolds only. Each file starts with a `WRITE:` /
`REFLECT ON:` comment block listing exactly what belongs in each section, and
every prose location is marked `% TODO (student): ...`.

| Folder | Report | Length |
|--------|--------|--------|
| `ch1/` | Dense networks on Fashion MNIST | 1 A4 |
| `ch2/` | Convolutions + regularisation on Fashion MNIST | 1 A4 |
| `ch3/` | Recurrent networks on SmartWatch gestures | 1 A4 |
| `ch4/` | Hypertuning a CNN on Flowers (Ray Tune) | max 3 A4 |

Figures live in `figures/` and are generated by running the notebooks / the Ch4
pipeline and the export step (see the repo root `README.md`). Until a figure
exists, the template renders a labelled placeholder box, so the report still
compiles.

## Before submitting

Fill in the header macros at the top of each `main.tex`:
`\studentnumber`, `\reportdate`, `\repourl` (and confirm `\studentname`).

## Compiling

With [tectonic](https://tectonic-typesetting.github.io/) (single binary):

```bash
tectonic reports/ch1/main.tex   # -> reports/ch1/main.pdf
```

Or with a TeX Live install:

```bash
cd reports/ch1 && latexmk -pdf main.tex
```

The figure paths are relative (`../figures/...`), so compile from inside each
chapter folder or pass the file path as shown above.
